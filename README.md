# ReactJS Tic-Tac-Toe

Simple Tic-Tac-Toe game base reactjs

## Running Tic-Tac-Toe

Run the game on development environment:

### `npm start`

## Running on production

In order to run this code on production, you must compile it:

### `npm build`
